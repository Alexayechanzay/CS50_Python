{m} m repetition

{m,n} m-n repetition (Lower_bound,upper_bound)

^   matches the start of the string

$   matches the end of the string or 
    just before the newline at the end of the string

[]  set of characters

[^m]    complementing the set (excluding m chars)
e.g [^arn] 
Returns a match for any character EXCEPT a, r, and n

[0-5][0-9] will match all the two-digits numbers from 00 to 59

\w  Returns a match where the string 
    contains any word characters 
    (characters from 
    a to Z, 
    digits from 0-9, 
    and the underscore _ character)
i.e \w = [a-zA-Z0-9_]

\b is defined as the boundary between a \w and a \W character 
(or vice versa), 
or between \w and the beginning or end of the string.

\W  not a word character

\d  decimal digits, = [0-9]
\D  not a decimal digits

\s  whitespace characters
\S  not a whitespace characters

|   same as (or) 
e.g A|B either A or B
